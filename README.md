######git的使用######

1.进入到项目目录
cd /F/学习/13其他职位技术/web前端开发/github/
git init --初始化，生成.git隐藏文件类似.svn

2.项目先clone下来
git clone git@github.com:gugujingling/fe.git

3.编辑or新增or删除

4.提交前，最好pull一下，保持与服务器版本一致，
git pull origin master

5.commit提交，写说明
git commit -m 'first_commit'

6.push上去
git push origin master
